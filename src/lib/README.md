# Resume Setup

## Quick Start

Simply place your `resume.pdf` file in the `public/` directory.

That's it! The website will automatically serve it at `/resume`.

## File Location

```
public/resume.pdf  ← Put your PDF here
```

## Features

- ✅ **Simple**: Just drop in a PDF file
- ✅ **Fast**: Served directly, no API needed
- ✅ **Reliable**: Works on any platform (Vercel, Netlify, etc.)
- ✅ **Clean UI**: Modern glassmorphism design
- ✅ **Download button**: Users can download your resume

## Updating Your Resume

1. Replace `public/resume.pdf` with your updated PDF
2. Commit and push
3. Done!

## Note

The PDF is served directly from the public folder for maximum reliability and performance.
